### 产品说明

目前市面绝对大多数网约车产品均已参考DD作为开发原型，滴滴在网约车的道路上一直没有找到更好的赢利点，MT加入网约车让整个出行形成了完整的商业形态，本团队在14年开始涉足网约车行业，为国内10多家网约车公司提供相应的技术支持。
目前中国网约车牌照有150家，但是真正运营起来的聊聊无几，大部分出租车公司拥有比DD更好的资源，但是相对来说技术较为落后。
